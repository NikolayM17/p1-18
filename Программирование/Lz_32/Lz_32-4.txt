Задание для контрольной работы (уровень "4")

Дан файл продаж автомобилей по датам месяцев input.txt.
Нужно получить суммарные ежемесячные и годовую продажи
Результат положить в файл output.txt

1. Загрузить входной файл в массив структур - функция load
2. Выгрузить результат в выходной файл - функция save

Проект, таким образом, состоит из 5 файлов
main.c
load.c
load.h
save.c
save.h

Пример файла input.txt
01.04.2010 Hyunday_Solaris 200
01.05.2010 Лада_Гранта 100
02.06.2010 Лада_Калина 200
02.04.2010 Hyunday_Solaris 300

Выходной файл
Апрель 2010 500
Май 2010 100
Июнь 2010 200
Итого за 2010 год 800

Ожидаемый результат
-------------------
Зип-архив с именем ЛЗ-24_Фамилия, где вместо слова "Фамилия" подставляется фамилия студента
Каждый файл в архиве имеет название по заданию, например, begin25.cpp
Каждый файл содержит
1. Комментарии
	- ФИО автора задачи
	- задание
	- ФИО автора решения
2. Код решения
3. Комментарий
	- вход (если пусто, просто слово "вход")
	- выход
Критерии оценки
---------------
9 задач - "5"
7 задач - "4"
5 задач - "3"
Примечание. Для оценок "3" и "4" обязательно наличие решений задач из каждого раздела
